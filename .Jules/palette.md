## 2024-05-27 - [Forms & Inputs Accessibility Polish]
**Learning:** For custom button groups or horizontal toggles (e.g., language selectors) formatted to look like standard controls, wrap them in an encompassing `role="group"` and label them appropriately (e.g., using `aria-labelledby`) instead of relying on disconnected `<label>` tags. Also, avoid committing lockfile updates unless explicitly intended, as running package installation can inadvertently regenerate the lockfile and pollute pull requests. Use `git restore pnpm-lock.yaml || true` or `git checkout -- pnpm-lock.yaml` during pre-commit steps to revert unwanted lockfile changes.
**Action:** Always wrap custom button groups in `role="group"` and label them appropriately. Always revert lockfile changes if they are not intended.
