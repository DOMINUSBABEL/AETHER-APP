## 2025-02-28 - Zodiac Selector Accessibility
**Learning:** Custom horizontally scrolling lists (like the Zodiac Selector in DailyScreen) are entirely skipped by keyboard navigation and provide zero context to screen readers without explicit ARIA roles and tab indices.
**Action:** When encountering a visual list that acts as a selector or provides information via active states, apply `tabIndex={0}` to the scrolling container to allow keyboard focus and arrow-key scrolling. Add `role="list"` to the container, `role="listitem"` to children, and `aria-current="true"` to indicate the currently active visual item.
